Программируем для Sony Smartwatch

Компания Sony выпускает телефоны, а также аксессуары к ним. И предоставляет к ним свой пакет разработки. Наиболее известный гаджет к телефону - это, пожалуй, умные часы Smartwatch.

Для установки SDK откройте диалоговое окно Window | Android SDK Manager. В меню окна выберите Tools | Manage Add-on Sites ….

Выберите вкладку User Defined Sites и щёлкните кнопку New…. В ведите в текстовом поле адрес http://dl-developer.sonymobile.com/sdk_manager/Sony-Add-on-SDK.xml и нажмите OK.

Закройте окно для ввода адреса. В окне SDK Manager в разделе Android 4.2.2 (API 19) отметьте флажком пункт Sony Add-on SDK 3.0, а в разделе Extras - Sony DeviceProfiles и установите пакет.

Отдельно можно скачать полную документацию по пакету.

После установки у вас появится новая папка в папке [директория Android SDK]\sdk\add-ons\addon-sony_add-on_sdk_3_0-sony-19. Вы можете импортировать любой проект для изучения. Например, SampleWidgetExtension. На данный момент компания сконцентрировалась на второй версии часов SmartWatch 2. Для часов первого поколения остались проекты BackwardsCompatibleControlPreferenceActivity и BackwardsCompatibleWidgetPreferenceActivity

Sony Smartwatch API работают только на устройствах под управлением Android 4.0 и выше. На устройстве надо установить приложение Sony SmartConnect (LiveWare manager), а также официальное приложение Sony SmartWatch. Если вы уже пользуетесь этими часами, то у вас это уже всё есть. Далее надо установить через блютуз соединение между часами и устройством и запускать своё устройство в режиме разработки, чтобы видеть результат.

Можно установить эмулятор часов. Для этого сначала установите в AVD Manager любое устройство от Sony, например, Xperia Z1, а затем в выпадающем списке Target выберите Sony Add-on SDK.

Импортируйте проекты из папки .../samples\SmartExtensions.

Проекты SmartExtensionAPI and SmartExtensionUtils являются библиотеками и их нужно включать в ваши приложения для SmartWatch.

Остальные проекты можно использовать как учебные пособия. Кроме того Sony выложило в открытый доступ исходный код двух своих программ Music player и 8 Game - Music player and 8 Game extensions for SmartWatch now available as open source – Developer World.
