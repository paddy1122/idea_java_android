Библиотека aFreeChart

Один из читателей прислал письмо с вопросом, знаком ли я с библиотеками, которые рисуют графики и диаграммы. Данные темы меня не особо интересовали, поэтому ответить было нечего. Для общего развития решил глянуть на одну из таких библиотек (на самом деле их достаточно много). Ниже мой результат краткого знакомства с библиотекой aFreeChart.

Официальная страница

Сама библиотека находится по адресу http://code.google.com/p/afreechart/. Скриншоты диаграмм и графиков можно увидеть на отдельной странице. Там же можно скачать саму библиотеку, исходники, файлы примеров. Также можно скачать документацию или просматривать её в онлайн-режиме.

Работаем с библиотекой

Итак, запустили проект, скопировали библиотеку, погладили кота.

Создадим класс, наследующий View. Для экономии я просто взял готовый код с сайта. Создайте новый класс и вставьте код с сайта.

Добавим в макет активности созданный View:


<?xml version="1.0" encoding="utf-8"?>

При попытке взглянуть на макет в режиме дизайна вылезли какие-то предупреждения. Не обращаем внимания, на работу не влияют.

Осталось написать код для активности. Я хочу видеть круговую диаграмму, которая наглядно показывает, кто лучше всех ловит мышей.


package org.afree.chart.demo;

import org.afree.chart.AFreeChart;
import org.afree.chart.ChartFactory;
import org.afree.chart.plot.PiePlot;
import org.afree.data.general.DefaultPieDataset;

import android.app.Activity;
import android.os.Bundle;

public class MainActivity extends Activity {

	@Override
	public void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);

		DefaultPieDataset dataset = new DefaultPieDataset();
		dataset.setValue("Мурзик", Integer.valueOf(20));
		dataset.setValue("Барсик", Integer.valueOf(10));
		dataset.setValue("Рыжик", Integer.valueOf(50));
		dataset.setValue("Васька", Integer.valueOf(10));
		dataset.setValue("Пушок", Integer.valueOf(10));

		AFreeChart chart = ChartFactory.createPieChart(
				"Победитель социалистического соревнования", dataset, true,
				false, false);
		PiePlot plot = (PiePlot) chart.getPlot();

		plot.setSimpleLabels(true);

		// plot.setLabelFont(new Font("SansSerif", Typeface.NORMAL, 8));
		// plot.setCircular(false);
		plot.setLabelLinksVisible(false);

		DemoView pieview = (DemoView) findViewById(R.id.demoView1);
		pieview.setChart(chart);
	}
}

Запускаем. Опа, и что мы видим. Рыжик - самая наглая рыжая морда из пушистой компании наловила больше всех мышей. Пойду угощу его молоком.
